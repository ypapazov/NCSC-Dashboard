# Fresnel Help

Welcome to the Fresnel cyber-incident coordination platform. Use this guide to get started and find answers about how the platform works.

## Sections

- [Getting Started](getting-started.html) — First login, navigation, and basic concepts
- [Events](events.html) — Creating, editing, and tracking cyber security events
- [TLP (Traffic Light Protocol)](tlp.html) — Understanding visibility levels for shared information
- [Roles & Permissions](roles.html) — What each role can do and how access is controlled
- [Dashboard](dashboard.html) — How the hierarchical status tree works
- [Campaigns](campaigns.html) — Grouping related events into campaigns

## Quick tips

- Use the **sidebar navigation** to move between sections of the platform.
- The **dashboard** gives you an at-a-glance view of the current situation across all sectors and organizations.
- Every event has a **TLP level** that controls who can see it — make sure you set the right one.
- If something doesn't look right, check whether your role grants access. See the [Roles](roles.html) page for details.
